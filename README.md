# Cloudflare vless 代理节点
### 1. 简介
利用Cloudflare的pages的全球CDN节点搭建vless类型VPN，包含美国、日本、中国台湾、中国香港、新加坡、马来西亚、韩国、俄罗斯等地节点
### 2. 代码改动
worker.js来自https://github.com/keyzf/cf-vless/tree/main
美化了界面，修改了部分内容，增加了许多新节点，目前有22个CDN节点
